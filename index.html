Setup Guide
1
Create free UploadThing account
Go to uploadthing.com → sign in with GitHub → click Create App. Name it anything. Then go to API Keys tab and copy your secret key.
2
Deploy to Vercel (free, 1 click)
Push these 3 files to a GitHub repo, then import it on vercel.com. Add one environment variable: 
UPLOADTHING_SECRET=your_secret_key_here
That's it. Vercel gives you a free *.vercel.app URL.
3
File structure in your repo
your-project/
├── index.html       ← this file
├── api/
│   ├── upload.js    ← serverless: get presigned URL
│   └── file.js      ← serverless: list your files
└── vercel.json      ← routing config
4
Done — open your Vercel URL and upload!
